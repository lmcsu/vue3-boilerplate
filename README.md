# Vue-3 boilerplate

## Base
- Vite
- TypeScript
- Element Plus
- Pinia
- Eslint
- vue-router
- normalize.css
- SCSS

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Previews production build
```
yarn preview
```

### Lints and fixes files
```
yarn lint
```
